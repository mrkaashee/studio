# Unified Image Editor

The `ImgStudio` component is the heart of our image editing suite. It provides a seamless, professional interface for a wide range of tasks, from simple cropping to complex annotations and filtering.

## Basic Usage

The full editor includes all tools enabled by default: Censor, Annotate, Aspect, Layers, Cropper, Filters, Transformations, and Resize.

## Custom Configurations

You can customize the editor by toggling specific features using the `props` attribute.

### Minimalist Editor
Only the cropper and basic transformations enabled.

::studio-view{name="editor"}
---
src: https://images.pexels.com/photos/1036622/pexels-photo-1036622.jpeg
props:
  censor: false
  annotate: false
  aspect: false
  layers: false
  filter: false
  resize: false
---
::

### Privacy-Focused View
Only the Censor (Blur/Pixelate) and Annotation tools enabled. Useful for redaction workflows.

::studio-view{name="editor"}
---
src: https://images.pexels.com/photos/3183153/pexels-photo-3183153.jpeg
props:
  cropper: false
  filter: false
  transform: false
  resize: false
  aspect: false
---
::

## Feature Breakdown

- **Censor Tool**: Quickly blur or pixelate sensitive information using multiple boxes.
- **Annotation Tool**: Add arrows, rectangles, and text with customizable colors and styles.
- **Cropper**: Flexible cropping with support for custom aspect ratios and shapes.
- **Filters & Presets**: Apply professional-grade CSS filters and predefined presets.
- **Transformations**: Precise rotation and flipping controls.
- **History & Undo/Redo**: Deep history tracking allows you to revert any change.
- **High-Quality Export**: Process and download your results entirely on the client side.

## Properties

| Property | Type | Default | Description |
| --- | --- | --- | --- |
| `src` | `string` | - | The URL or base64 of the image to edit. |
| `censor` | `boolean` | `true` | Enable/disable the Censor tool. |
| `annotate` | `boolean` | `true` | Enable/disable the Annotation tool. |
| `cropper` | `boolean` | `true` | Enable/disable the Cropper tool. |
| `filter` | `boolean` | `true` | Enable/disable filters and presets. |
| `transform` | `boolean` | `true` | Enable/disable rotation and flip tools. |
| `resize` | `boolean` | `true` | Enable/disable image resizing. |
| `history` | `boolean` | `true` | Enable/disable undo/redo functionality. |
